# 🧮 Modern Calculator Website

A sleek and responsive web-based calculator built using **HTML, CSS, and JavaScript**. This project provides basic arithmetic functionality with a modern glassmorphism UI design.

---

## 🚀 Features

* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (−)
  * Multiplication (×)
  * Division (÷)
* Clear display functionality
* Support for decimal values
* Double zero (`00`) input
* Error handling for invalid expressions
* Responsive and modern UI (glassmorphism style)

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and layout (including gradients and glass effects)
* **JavaScript (Vanilla)** – Functionality and logic

---

## 📁 Project Structure

```
calculator-project/
│── index.html      # Main HTML structure
│── style.css       # Styling and layout
│── script.js       # Calculator logic
```

---

## 📸 How It Works

* Users click buttons to input numbers and operators.
* Input is displayed on the screen.
* Pressing `=` evaluates the expression using JavaScript.
* Errors are handled and displayed as `"Error"`.

---

## ⚙️ Functions Overview

### `append(value)`

Adds numbers or operators to the display.

### `clearDisplay()`

Clears the input field.

### `calculate()`

Evaluates the mathematical expression using `eval()`.

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start calculating!

---

## ⚠️ Note

* This project uses JavaScript's `eval()` function, which is fine for learning purposes but **not recommended for production apps** due to security risks.
